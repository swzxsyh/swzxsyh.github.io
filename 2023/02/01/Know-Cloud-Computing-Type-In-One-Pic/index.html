一图了解云服务类型Iaas/SaaS/Paas

发表于 2023-02-01

Cloud Computing

云计算
On Premises Application 应用 SaaS (Software as a Service)
Data 数据
Runtime 运行时 PaaS (Platform as a Service)
Middleware 中间件
OS
Virtualization 虚拟化 Iaas (Infrastructure as a Service)
Server 服务器
Storage 存储
Network 网络

云计算提供功能

  • Application 应用
    • 用户使用Data 数据的应用
  • Data 数据
    • 企业直接区分点，不同企业数据不同
  • Runtime 运行时
    • 取决于应用运行在什么框架、编程语言上
  • Middleware 中间件
  • OS
  • Virtualization 虚拟化
    • 将一台Server分割多个虚拟部分，提供不同服务
  • Server 服务器
  • Storage 存储
    • 数据存储硬件
  • Network 网络

On Premises

自建IDC，相对应云计算的结构

Iaas

用户可以自行提供OS-Application的能力，云供应商只需要提供基础服务

  • Virtualization 虚拟化
  • Server 服务器
  • Storage 存储
  • Network 网络

这部分功能供应商接管后提供服务，对于用户而言 用户与跟服务器硬件无关，当作一个Service使用，因此被称为Infrastructure as a Service

Paas

用户只关心业务，不关心底层，自行进行业务功能设计构建，云供应商需要提供

  • Runtime 运行时
  • Middleware 中间件
  • OS
  • Virtualization 虚拟化
  • Server 服务器
  • Storage 存储
  • Network 网络

Saas

用户直接使用云供应商集成功能的服务，自己直接使用

如Office 365、远程桌面Citrix功能
