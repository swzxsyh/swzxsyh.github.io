3.3 SDK使用

软件开发工具包(Software Development Kit )包含 jar包、API文档、工具类的集合都可以叫做 SDK

  • 请求参数

    参数名称 参数类型 是否必须 样例取值
    phoneNumbers 手机号 String 必须 15000000000
    signName 签名 String 必须 云通信
    templateCode 模板 String 必须 SMS_0000
    TemplateParam String 可选 {“code”:”1234”,
    “product”:”ytx”}

  • 响应参数

    参数名称 出参类型 样例取值 参数说明
    RequestID String 8906582E-6722 请求ID
    Code String OK 状态码-返回OK代表请求成功，嵌套错误代码详见错误码列表
    Message String 请求成功 状态码的描述
    Bizld String 134534^4351232 发送回执ID，可根据ID查询具体的发送状态

    SmsUtils.java放入Utils
