一.Vue基础

1.1 简介

是一个JavaScript框架，可以简化Dom操作，响应式数据驱动

官网： https://cn.vuejs.org

1.2 快速入门

1.3 el:挂载点

  • Vue实例范围

    Vue会管理EL选项 命中的元素 和其内部的 后代元素

  • 其他选择器

    可以使用其他选择器，如class、标签选择器，但是建议使用id选择器，更清晰

  • 其他DOM元素

    可以设置其他双标签，但是不能使用HTML和body标签

1.4 data数据对象

Vue用到的数据定义再Data中

data中可以写复杂类型数据

渲染复杂类型数据时，遵守js语法即可

2.1 内容绑定，事件绑定

  • v-text

    • 作用：设置标签的文本值(textContent)

      默认写法会替换全部内容，使用差值表达式<!–￼25–>可以替换指定内容(推荐差值表达式方法)

    • 案例

  • v-html

    • 作用：设置标签的innerHTML

      内容中有html结构均会被解析为标签

      v-text指令无论什么结构都只会解析为文本

    • 案例

  • v-on基础

    • 作用：为元素绑定事件

      事件名不需要写on

      指令可以简写为@

      绑定的方法定义在methods中

      方法内部通过this关键字可以访问定义在data中的数据

    • 语法

    • 案例

2.1.1 综合案例：计数器

  • 分析

    data中定义数据

    methods添加add,minus方法

    使用v-text将num给span设置标签

    使用v-on将add和minus分别设置给+-按钮

    累加的逻辑：小于10可加，否则alert

    累减的逻辑：大于0可减，否则提示

  • 实现

2.1.2 小结
  • 创建Vue实例时：el挂载点，data数据，methods方法
  • v-on指令的作用是绑定事件，简写为@
  • 方法中通过this关键字获取data中的数据
  • v-text指令的作用是：设置元素的文本值，简写为<!–￼26–>

2.2 显示切换，属性绑定

  • v-show

    • 作用：根据表达式的真假，切换元素的显示和隐藏

      原理是修改元素的display状态，实现显示/隐藏

      指令后的元素，最终都会解析成为boolean值

      值为true显示，值为false隐藏

    • 案例

  • v-if

    • 作用：根据表达式的真假，切换元素的显示和隐藏(操纵dom元素)

      本质是通过操纵DOM元素来切换显示状态

      表达式的值为true，元素存于DOM树中，false则从DOM树移除

      频繁切换使用v-show，反之使用v-if，前者切换消耗小

    • 案例

  • v-bind

    • 作用：设置元素的属性(如：src/title/class等)

      完整语法：v-bind:属性名

      简写可以省略v-bind，只保留:属性名

      需要动态的增删class建议使用对象方式

    • 语法

    • 案例

2.2.1 综合案例：图片轮播

    定义图片数组

    添加图片索引

    绑定src属性：v-bind

    图片切换逻辑：v-on

    显示状态切换：v-show

  • 实现

2.2.2 小结
  • 列表数据用数组保存
  • v-bind可以设置元素属性，如class/src
  • v-show和v-if都可以切换元素状态，频繁切换应使用v-show

      绑定的数据 与 表单元素的值 相互绑定

四.综合应用：网络播放器

4.1 需求分析

  • 歌曲搜索

    按下回车(v-on ，.enter)

    查询数据(axios 接口 v-model)

    渲染数据(v-for 数组 that)

    • 搜素接口：

    请求地址：https://autumnfish.cn/search

    请求方法：get

    请求参数：keywords

    响应内容：歌曲搜索结果

    • 注意

      服务器返回数据，如果比较复杂，获取时需注意层级结构

      通过审查元素，快速定位到需要操纵的元素

  • 歌曲播放

    点击播放(v-on 自定义参数)

    歌曲地址获取(接口 歌曲ID)

    歌曲地址设置(v-bind)

    • 歌曲URL接口获取

      请求地址：https://autumnfish.cn/song/url

      请求方法：get

      请求参数：id

      响应内容：歌曲URL地址

    • 注意

      歌曲id依赖歌曲搜索的结果，对于不用的数据也需要关注

  • 歌曲封面

    点击播放(增加逻辑)

    歌曲封面获取(接口 歌曲id)

    设置歌曲封面(v-bind)

    • 歌曲详情获取

      请求地址：https://autumnfish.cn/song/detail

      请求方法：get

      请求参数：ids

      响应内容：歌曲详情，包含封面信息

    • 注意

      在Vue中通过v-bind操作元素属性，这里主要指src属性

  • 歌曲评论

    点击播放(增加逻辑)

    歌曲评论获取(接口 歌曲id)

    渲染歌曲评论(v-for)

    • 热门评论获取

      请求地址：https://autumnfish.cn/comment/hot?type=0

      请求方法：get

      请求参数：id

      响应内容：歌曲的热门评论

    • 注意

      在Vue中通过v-for获取生成列表

  • 播放动画

    监听音乐播放(v-on play)

    监听音乐暂停(v-on pause)

    操纵类名(v-bind 对象)

    • 注意

      audio标签的play事件会在音频播放时触发

      audio标签的pause事件会在音频暂停时触发

      通过对象的方式设置类名，类名生效与否取决于后面值的真假

  • mv播放

    mv图标显示(v-if)

    mv地址获取(接口 mvid)

    遮罩层(v-show v-on)

    mv地址设置(v-bind)

    • MV地址获取

      请求地址：https://autumnfish.cn/mv/url

      请求方法：get

      请求参数：id

      响应内容：MV地址

    • 注意

      不同接口需要的数据是不同的，文档阅读需要仔细

      页面结构复杂之后，通过审查元素的方式快速定位相关元素

      响应式数据，一定要在data中定义

4.2 代码实现

4.3 小结

  • 服务器返回数据，如果比较复杂，获取时需注意层级结构
  • 通过审查元素，快速定位到需要操纵的元素
  • 歌曲id依赖歌曲搜索的结果，对于不用的数据也需要关注
  • 在Vue中通过v-bind操作元素属性
  • 在Vue中通过v-for获取生成列表
  • audio标签的play事件会在音频播放时触发
  • audio标签的pause事件会在音频暂停时触发
  • 通过对象的方式设置类名，类名生效与否取决于后面值的真假
  • 不同接口需要的数据是不同的，文档阅读需要仔细
  • 页面结构复杂之后，通过审查元素的方式快速定位相关元素
  • 响应式数据，一定要在data中定义
