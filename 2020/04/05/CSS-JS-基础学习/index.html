一.CSS概述
层叠样式表
作用：美化页面
层叠样式：对同一个标签添加多个不同对样式，所有的样式会叠加在一起显示出效果

优点:
•实现了样式和内容对分离，提高了显示效果和样式的复用性
•降低耦合性，分工更加明确，CSS专门用于美化，HTML专门用于结构搭建
